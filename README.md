# Full-Stack-Challenges
This is a series coding practices for full-stack applications, which cover HTML, CSS, JavaScript, ReactJS and MongoDB.

## Installation
`$git clone https://github.com/cacijiang/Full-Stack-Challenges.git`

For some projects, maybe you should install packages by using:`$npm install`

Before run the projects, you can try:
`$npm run jshint` or `$npm test`
